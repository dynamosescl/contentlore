Methodology

How We Track

ContentLore polls Twitch Helix and Kick every 15 minutes via a Cloudflare Worker scheduler, capturing live status, viewer counts, titles, and tags for every tracked creator. Snapshots are stitched into stream sessions for historical analysis.

Scene detection groups concurrent streams by server using title keyword matching against our UK server registry. The beef and lore editorial layers are manually curated with beat-by-beat timelines and chapter breakdowns.
